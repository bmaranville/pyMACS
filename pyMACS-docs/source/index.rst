**pyMACS** Documentation
===========================

**pyMACS** is a python library designed specifically to interface between McStas Monte-Carlo simulations of the MACS instrument at NIST and user friendly python tools. A number of options are available to perform the virtual experiment. Detailed examples of usage can be found in the jupyter notebooks on the github page. Documentation of the functions within pyMACS itself is contained here. 

.. note:: 

This project will receive very few updates after release, but the source is readily available and can be modified at any time.
